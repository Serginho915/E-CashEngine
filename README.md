# e-CashEngine

Fullstack online income content platform scaffold for daily AI-assisted articles by Irina Vovk.

Folders:
- `frontend` — Next.js + TypeScript + Tailwind + SCSS
- `backend` — Node.js + Express API (TypeScript)

Run:

1. Install dependencies in each folder:

```bash
cd frontend && npm install
cd ../backend && npm install
```

2. Start backend and frontend in separate terminals:

```bash
cd backend && npm run dev
cd frontend && npm run dev
```
